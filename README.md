# grunt-beep

I present to you the best Grunt task ever! It beeps!

## install it!

```
npm install grunt-beep --save-dev
```

## load it!

```js
grunt.loadNpmTasks('grunt-beep');
```

## use it!

```js
grunt.registerTask('default', ['jshint', 'nodeunit', 'beep']);
```
